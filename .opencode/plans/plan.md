# CompaMarketer - Plan SSD

## Visión General
Aplicación web React para comparar precios de productos en supermercados chilenos (Líder, Unimarc, Santa Isabel, Jumbo, Spid).

## Fase 0: Dependencias
recharts, framer-motion, react-icons

## Fase 1: Base de Datos Mock (JSON)
src/data/productos.json con precios actuales e históricos

## Fase 2: Componentes
Header, ProductList, ProductCard, ProductModal, PriceComparisonTable, PriceHistoryChart, SupermarketBadge, OfferBadge

## Fase 3: Visual Design
Paleta: #FF6B35 (naranja), #004E89 (azul), #1A936F (verde)
Tipografía: Poppins
Animaciones: Framer Motion
Responsive: Mobile-first

## Fase 4: Funcionalidades
- Catálogo con filtro por categoría
- Modal con tabla comparativa + gráfico + links
- Highlight al precio más barato
- Badge de oferta
- Link externo al producto

## Fase 5: Verificación
npm start
